System zarządzania sesjami terapeutycznymi

1. Klucz Anthropic API

Do generowania notatek AI. Zapisywany tylko lokalnie na urządzeniu.

Pobierz klucz na console.anthropic.com → API Keys. Koszt: ~0,25 zł / sesję (model Haiku).

2. Supabase — sync między urządzeniami

Opcjonalne. Pozwala używać aplikacji na telefonie i komputerze jednocześnie.

1. Wejdź na supabase.com → New project (darmowy)
2. SQL Editor → wklej kod → Run
CREATE TABLE IF NOT EXISTS clients ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), name text NOT NULL, age int, issue text, notes text, created_at timestamptz DEFAULT now() ); CREATE TABLE IF NOT EXISTS sessions ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), client_id uuid REFERENCES clients(id) ON DELETE CASCADE, date timestamptz DEFAULT now(), duration_sec int DEFAULT 0, transcript text DEFAULT '', notes jsonb, free_notes text DEFAULT '', organized_notes text DEFAULT '', mood int DEFAULT 5, tags jsonb DEFAULT '[]', created_at timestamptz DEFAULT now() ); ALTER TABLE clients ENABLE ROW LEVEL SECURITY; ALTER TABLE sessions ENABLE ROW LEVEL SECURITY; DO $$ BEGIN IF NOT EXISTS (SELECT 1 FROM pg_policies WHERE tablename='clients' AND policyname='all') THEN CREATE POLICY "all" ON clients FOR ALL USING (true); END IF; IF NOT EXISTS (SELECT 1 FROM pg_policies WHERE tablename='sessions' AND policyname='all') THEN CREATE POLICY "all" ON sessions FOR ALL USING (true); END IF; END $$;
3. Settings → API → skopiuj Project URL i anon key poniżej
